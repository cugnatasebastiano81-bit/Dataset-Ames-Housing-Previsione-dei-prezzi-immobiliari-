# Dataset-Ames-Housing-Previsione-dei-prezzi-immobiliari-
Questo notebook implementa una pipeline di Machine Learning modulare e riproducibile per la previsione del prezzo di vendita di immobili residenziali nella città di Ames (Iowa), a partire dalle loro caratteristiche fisiche, qualitative e di contesto. L'intero lavoro è organizzato in fasi indipendenti e documentate.
